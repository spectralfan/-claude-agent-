# JChatMind UI — 前端界面

> React 19 + TypeScript + Vite + Ant Design X

## 页面路由

| 路径 | 说明 |
|------|------|
| `/chat` / `/chat/:chatSessionId` | 普通对话 |
| `/coding` / `/coding/:sessionId` | AI Coding 工作台 |
| `/knowledge-base` | 知识库管理 |

## 开发

```bash
npm install
npm run dev     # 开发服务器 :5173
npm run build   # 构建到 dist/
```
